# House Price Prediction System

## Project Overview

This project is an end-to-end Machine Learning application that predicts the selling price of residential houses based on their characteristics.

The project is being developed as part of a client/internship project and follows the complete Machine Learning lifecycle from data understanding to cloud deployment.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- FastAPI
- Streamlit
- Docker
- Git
- GitHub
